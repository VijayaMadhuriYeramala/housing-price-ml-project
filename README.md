
# 🏠 Housing Price Prediction (ML Project)

## 📌 Overview

Built a machine learning model to predict housing prices using California housing dataset.

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn

## 🤖 Models Used

* Linear Regression
* Random Forest Regressor
* Pipeline (Imputer + Model)

## 📊 Results

* Linear Regression R2: ~0.62
* Random Forest R2: ~0.81
* Pipeline R2: ~0.818

## 💡 Key Learnings

* Feature vs target (X and y)
* Handling missing data
* Model training and evaluation
* Improving models
* Building ML pipeline
